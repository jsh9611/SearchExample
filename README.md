# Search Example

책을 검색하고 북마크를 할 수 있습니다.

## 기술 스택

- Swift UI
- Core Data
- iOS 14
- Xcode 14.2

## 실행 방법
1. 다운로드 방법
  - git clone https://github.com/jsh9611/SearchExample.git
2. 실행 방법
  - Xcode에서 Build 하면 됩니다.
